## Who am I? 🤔

I am an **Android Developer specialized in Kotlin**, focused on building **robust, scalable, and production-ready mobile applications**.  
I mainly work on systems where **stability, security, performance, and code quality** are critical.

I am part of the development of **Phoenix**, an Android-based **access control system** for buildings and corporate environments. The application integrates advanced features such as **SIP calls, DTMF signaling, PIN-based access, QR authentication, and facial recognition**, following **Clean Architecture and MVVM** principles.

While Kotlin is my primary tool, my real strength lies in **designing solid architectures** and **transforming complex technical requirements into reliable and maintainable solutions** for real-world use cases.

## Featured Projects 🚀

### 🔐 Phoenix – Access Control System
Android application for **building and corporate access management**, designed for real-world production environments and dedicated hardware.

**Key features:**
- **SIP-based calling** and intercom integration  
- **DTMF signaling** for remote access control  
- Authentication via **PIN, QR codes, and facial recognition**  
- **Biometric authentication** support  
- **Clean Architecture + MVVM**  
- Integration with external hardware modules and remote backend systems  

This project is heavily focused on **stability, security, and long-term maintainability** in production deployments.
